# The MakSoil-Sensing Project
Collects available NPK values and Logs them on a remote Database for visualisation, analysis to interprete and use in model training
## FEATURES
1. RS485 Module
2. LILYGO SIM7000G ESP32 board
3. RS-7-in-1 soil sensor probe
4. MTN SIM card
5. 18650 rechargable battery
6. 18650 v3 battery charge shield bord
